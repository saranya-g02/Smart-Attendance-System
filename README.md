# Restaurant Data Analysis

## Cognifyz Technologies Data Analysis Internship Project

### Project Description

This project was completed as part of the **Cognifyz Technologies Data Analysis Internship**. It focuses on performing **Exploratory Data Analysis (EDA)** on a restaurant dataset to uncover valuable business insights related to cuisines, cities, restaurant ratings, pricing, customer engagement, online delivery, table booking, restaurant chains, and geographical distribution.

The analysis was performed using **Python** in **Jupyter Notebook**, with visualizations created using Matplotlib and Seaborn.

---

#  Objectives

* Perform data cleaning and preprocessing.
* Explore restaurant distribution across different cities.
* Analyze the popularity of cuisines.
* Study aggregate ratings and customer voting patterns.
* Analyze restaurant price ranges.
* Compare online delivery and table booking services.
* Identify popular restaurant chains.
* Visualize restaurant locations geographically.
* Generate meaningful business insights using data visualization.

#  Key Insights

* North Indian cuisine is the most common cuisine.
* New Delhi has the highest number of restaurants.
* Most restaurants belong to Price Range 1.
* Restaurants offering online delivery generally receive higher ratings.
* Higher-priced restaurants are more likely to provide table booking.
* Premium restaurants tend to achieve better average ratings.
* Popular restaurant chains maintain strong customer engagement and consistent ratings.
* There is a weak to moderate positive correlation between customer votes and aggregate ratings.
